# Personal Profile Page Website

## Description
This is a responsive personal profile page created using HTML, CSS, and JavaScript.
It showcases personal information, skills, experience, and contact details.

## Features
- Responsive layout
- Dark theme with red accents
- Smooth scrolling animation
- Clean and well-commented code

## How to Run
1. Download and extract the zip file
2. Open index.html in a web browser
